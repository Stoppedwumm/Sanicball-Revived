Sanicball - Revived

Welcome to the Sanicball documentation. This project is a revival of the original Sanicball game, which is a fast-paced racing game featuring characters from the Sanic meme universe.

Getting Started

To get started with Sanicball, you can clone the repository from GitHub and follow the setup instructions provided in the README file.
